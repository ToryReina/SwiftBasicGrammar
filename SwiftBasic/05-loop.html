循环

  • OC for循环
/*
    OC的for:
        for (int i = 0; i < 5; i++) {
            NSLog(i)
        }
*/
  • swfit for 常规写法
for var i = 0; i < 5; i++ {
    print(i)
}

  • swift 常用写法

    // swift 常用写法
for i in 0...5 {
  print(i)
}

  • 范围运算符

    // 范围运算符
// ... 闭合范围运算符 表示 a 到 b, [a, b] 0-5, 包含a,也包含b
// ..< 半闭合范围运算符 表示 a 到 b, [a, b) 0-4, 包含a,但是不包含b

  • 省略下标

    • _ 表示忽略对应位置的值
      for _ in 0...5 {
print("hello")
