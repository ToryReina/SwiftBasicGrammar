变量和常量

  • let 定义常量，一经赋值不允许再修改
  • var 定义变量，赋值之后仍然可以修改
/*
    OC中:
        定义变量: int a = 1;
        定义常量: int const a = 1;

    swift:
        定义常量:   let 常量名 = 值
        定义变量:   var 变量名 = 值

    var 修饰的是可变的
    let 修饰的是不可变的

*/

// 定义变量
var m = 20

// 修改变量
m = 25

print("m = \(m)")

// 定义常量
let n = 10

print("n =  \(n)")
// 常量初始化后不能再修改
//n = 15
